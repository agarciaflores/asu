README.txt

This folder contains all of the code used for the project. The folders 'qkd-code-and-data', 'entanglement-code-and-data', and 'teleportation-code-and-data' each contain the files necessary to produce the data in our presentation. For these folders, it is assumed that the user is using the IBM Quantum Lab: https://quantum.ibm.com/lab.

The folder analysis-of-data contains all of the code necessary to produce figures from our data.
